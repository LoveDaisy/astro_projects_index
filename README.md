# Astro tools index
This is an index for astro tools developed by a group of amateur astronomers. We develop these tools for our interest.

| project | owner | description | language |
|---------|-------|-------------|----------|
| [star_alignment](https://github.com/LoveDaisy/star_alignment) | [LoveDaisy (章佳杰)](https://github.com/LoveDaisy) | Align several astronomy / nightscape images by star points and average them to reduce noise. Apply to wide angle lens as well as tele-lens. | python |
| [ice_halo_sim](https://github.com/LoveDaisy/ice_halo_sim) | [LoveDaisy (章佳杰)](https://github.com/LoveDaisy) | A ray-tracing based simulation program for ice halos. | c++ |
